## 🧠 What is Vectra?

Vectra is a **personal AI strategy assistant** that helps you decide what to build.

It doesn't just generate ideas. It evaluates them, challenges them, and gives you a clear direction.

Every analysis ends with a **verdict**:

```
EXECUTE
RESTRUCTURE
KILL
```


---

## ⚙️ What it does

* Breaks down your idea into core assumptions
* Identifies weaknesses and risk points
* Rejects weak directions
* Selects the strongest execution path
* Generates a structured execution plan
* Assigns a clear, decisive verdict

## 🧾 Permanent Report Storage

## 🧩 Analysis Fingerprint

Each report includes a unique verification hash.

Ensures integrity — the report cannot be altered unnoticed.

## 🖥️ How it works
1. Connect wallet
2. Enter your idea
3. Run analysis
4. Receive verdict
5. Save report permanently
6. Continue execution via Telegram

---

## 🌐 Tech Stack
* Frontend: Next.js + Tailwind CSS
* AI Inference: Qwen 3.5 (Nosana decentralized GPU network)
* Storage: IPFS (Pinata)
* Identity: Solana Wallet (Phantom)
* Agent Extension: Telegram Bot
